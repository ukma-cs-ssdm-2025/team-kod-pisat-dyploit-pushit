1) Як користувач, я хочу шукати фільми за назвою чи жанром, актором чи режисером, щоб швидко знаходити те, що мене цікавить.
   Критерії прийняття:

   Given: користувач відкриває головну сторінку сайту
   When: вводить у пошук назву фільму, актора, жанр або режисера
   Then: система показує список релевантних результатів

   Given: користувач вводить неіснуючу назву
   When: натискає “Пошук”
   Then: система показує повідомлення “Нічого не знайдено”

INVEST оцінка:

| Критерій | Виконано | Коментар |
|-----------|-----------|----------|
| I (Independent) | Історія не залежить від інших, описує окрему функцію пошуку |
| N (Negotiable)  | Можна уточнювати алгоритм пошуку, не змінюючи мету |
| V (Valuable)    | Дає користувачу користь — можливість знаходити потрібні фільми |
| E (Estimable)   | Обсяг роботи чітко визначений (створення пошуку + обробка помилок) |
| S (Small)       | Невелика, поміщається у спринт |
| T (Testable)    | Має Given–When–Then сценарії |
